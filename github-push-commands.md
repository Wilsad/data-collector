# GitHub Push Commands

## After creating your repository on GitHub, run these commands:

### 1. Add Remote Repository
```bash
cd "c:\Users\Wilad\CascadeProjects\windsurf-project\data-collection-app"
git remote add origin https://github.com/YOUR_USERNAME/data-collector.git
git branch -M main
```

### 2. Push to GitHub
```bash
git push -u origin main
```

### 3. Enable GitHub Pages
1. Go to your repository on GitHub
2. Click "Settings" tab
3. Click "Pages" in left menu
4. Source: "Deploy from a branch"
5. Branch: "main" and "/ (root)"
6. Click "Save"

### 4. Your Site Will Be Live At:
https://YOUR_USERNAME.github.io/data-collector/

## Replace YOUR_USERNAME with your actual GitHub username
